BUILD
=====

You'll need:
  Qt 5.5  (for module :webkit)
  Qt 5.8+ (for module :webengine)

- compile module :webkit with Qt 5.5
- compile ./lib/* with Qt 5.5
- compile EQL5 (including module :webengine) with Qt 5.8+
